# Shree GuruDatt Properties

A modern real estate properties website built with React, TypeScript, Vite, and Tailwind CSS.

## Project Overview

**Shree GuruDatt Properties** is a real estate platform helping clients find residential and commercial properties in Goregaon.

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- React Router DOM

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

Follow these steps:

```sh
# Step 1: Navigate to the project directory.
cd shree-GuruDatt-properties

# Step 2: Install the necessary dependencies.
npm install

# Step 3: Start the development server with auto-reloading and an instant preview.
npm run dev
```

### Building for Production

```sh
# Build the project
npm run build
```

### Testing

```sh
# Run tests
npm run test

# Run tests in watch mode
npm run test:watch
```
